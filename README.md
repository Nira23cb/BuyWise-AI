# 🛍️ BuyWise AI

An AI-powered shopping assistant that helps users make informed purchasing decisions by analyzing product reviews and generating concise AI-driven insights.

---

## ✨ Features

- Analyze products using product links
- AI-generated product summary
- Pros & Cons extraction
- Buy Score
- Smart purchase recommendation
- Responsive and user-friendly interface

---

## 🛠️ Tech Stack

- Java
- Spring Boot
- HTML
- CSS
- JavaScript
- Groq API (Llama Model)

## 🚀 Getting Started

1. Clone the repository.
2. Configure your Groq API key.
3. Run the Spring Boot application.
4. Open the application in your browser.

---

## 🔒 Security Note

For security reasons, API keys are not included in this repository. Configure your own Groq API key before running the project.
